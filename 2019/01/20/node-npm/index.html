package.json

  • 建议每个文件都有这个文件（包描述文件）
  • 这个文件可以用过npm init来创建
  • 执行npm install加上–save用来保存依赖项信息
  • npm install 会把所有依赖项都加载回来

npm常用命令

  • npm init

  • npm install 包名

  • npm uninstall 包名

  • npm install --save

  • npm help
